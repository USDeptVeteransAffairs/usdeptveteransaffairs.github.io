Welcome to the VA GitHub page

The US Department of Veterans Affairs is committed to collaboratively working with the open source community to develop better software applications, tools, and services in support of veterans. This page provides information on some of our most important projects as well as an index to all VA source code repositories available on GitHub.

VistA 3: Current VistA

OSEHRA VistA

This Open Source Electronic Healthcare Record Alliance (OSEHRA) product is based upon FOIA VistA and contains additional enhancements and improvements from the Open Source Community. The code here is continuously monitored and utilized by a wide range of community members using software quality testing methods and agile software development practices. This is the primary code repository to use for working with both the VA and the open source community.

  • M-Components
  • Patch Stream
  • Forum Patch Module

FOIA VistA

This repository contains VA VistA as released under the Freedom of Information Act (FOIA), and certified by OSEHRA for use by the Open Source community.

  • M-Components
  • Patch Stream

VistA Immunization Project

This repository contains source code and documentation for the collaboration between VA and OSEHRA community developers to enhance the Immunization module of the Patient Care Encounter package.

GI Bill Comparison Tool

This is a web application for veterans to calculate their educational benefits and research approved programs. The tool makes it easier for Veterans, Servicemembers and dependents to calculate their Post-9/11 GI Bill benefits and learn more about VA’s approved colleges, universities and other education and training programs across the country.

VistA NOVO

The goal of this project is to allow developers to use and contribute to VistA capabilities using mainstream architectural approaches and modern programming languages. Here you will find test and demonstration code for the VistA Novo SDK.

VA Center for Innovation Projects

AWARE: Alert Watch and Respond Engine

An automated tool enabling clinicians, supervisors, and administrations to monitor and track alert responses in CPRS.

COMS: Chemotherapy Ordering Management System

A web-based application providing oncology teams with ordering, preparation, and documentation of chemotherapy.

RAPTOR: Radiology Protocol Tool Recorder

An automated, electronic tool allowing radiologists to optimize advanced medical imaging protocols.
